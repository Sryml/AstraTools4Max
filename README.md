3DStudio MAXScript Utility to modify the Blade of Darkness model.
3DStudio Max 2.5 or higher compatible
- 3ds Max 2.5✅
- 3ds Max 2012 64-bit✅

Copyright (C) Dec 2021 Sryml

## Usage

Copy the contents of the "3ds Max" folder to `3DSMAX2.5` or `3ds Max xxxx`
